HTML Attributes

HTML attributes provide additional information about elements in an HTML document.

Common HTML Attributes

  • id: Specifies a unique id for an element.
  • class: Specifies one or more class names for an element (used for CSS and JavaScript).
  • style: Specifies inline CSS styles for an element.
  • title: Provides additional information about an element (displayed as a tooltip).
  • href: Specifies the URL of a link (used in <a> tags).
  • src: Specifies the source of an image or script (used in <img> and <script> tags).
  • alt: Provides alternative text for an image (used in <img> tags).
